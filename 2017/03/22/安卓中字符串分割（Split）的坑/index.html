安卓中字符串分割（Split）的坑

发表于 2017-03-22 | 分类于 Android ， Android 实战

版权声明：本文为 wintersmilesb101 -（个人独立博客– http://wintersmilesb101.online 欢迎访问）博主原创文章，未经博主允许不得转载。

今天在做一款音乐APP的时候需要解析JSON数据，解析到最后有个数组由||分割开来，于是想到使用字符串的spilt()方法来做，

说做就做：s.spilt(“||”);

运行，居然没用，没有任何分割的字符串出现。

折腾了许久，通过查询资料，终于找到问题，原来是在安卓下使用spilt()方法的时候需要加[]符号。

比如，分割字符串还是我这里的||，

我们正确的打开方式应该是：

s.spilt(“[||]”);

运行一下，完美。
