2.在引用的同时可以对 Nest 进行相关的配置

  • color ： 线条颜色, 默认: ‘0,0,0’ ；三个数字分别为(R,G,B)，注意用,分割

  • opacity : 线条透明度（0~1）, 默认: 0.5

  • count : 线条的总数量, 默认: 150，（建议修改少点，否者 cpu 占用过高）

  • zIndex : 背景的z-index属性，css属性用于控制所在层的位置, 默认: -1
